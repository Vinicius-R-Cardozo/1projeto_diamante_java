# 🏥 API de Gerenciamento de Médicos e Pacientes

API REST desenvolvida com **Java + Spring Boot** para gerenciamento completo de médicos e pacientes.

---

## 📌 Sobre o projeto

Este projeto consiste em uma API que permite realizar operações de CRUD (Create, Read, Update, Delete) para médicos e pacientes, seguindo boas práticas de desenvolvimento backend.

---

## ⚙️ Funcionalidades

✔ Cadastro de médicos e pacientes  
✔ Listagem de registros  
✔ Atualização de informações (PUT)  
✔ Remoção de dados (DELETE)  

---

## 🛠️ Tecnologias utilizadas

- ☕ Java  
- 🚀 Spring Boot  
- 📦 Spring Data JPA  
- 🛠️ Maven  
- 🗄️ Hibernate  

---

## 🧠 Conceitos aplicados

- 📌 Arquitetura em camadas:
  - Controller
  - Service
  - Repository  

- 📌 Uso de `ResponseEntity` para controle das respostas da API  

- 📌 Status HTTP corretamente aplicados:
  - `200 OK`
  - `201 Created`
  - `204 No Content`
  - `404 Not Found`  

- 📌 Manipulação de dados com JPA/Hibernate  

- 📌 Boas práticas no desenvolvimento de APIs REST
